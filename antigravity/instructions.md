# Objetivo del proyecto

Crear una aplicación de seguimiento de anime y manga llamada Anime Tracker, usando NextJS (App Router) y Tailwind.

El proyecto ya está creado y con tailwind instalado, pero sin estilos ni funcionalidades. El objetivo es implementar el diseño y las funcionalidades básicas de una app moderna, premium y minimalista, siguiendo exactamente los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta de "Antigravity" dentro del proyecto.

# Funcionalidades

- Autenticación de usuarios (login, registro, logout)
- Dashboard con estadísticas del usuario
- Listado de anime y manga
- Detalle de anime y manga
- Búsqueda de anime y manga
- Filtros de anime y manga
- Ordenamiento de anime y manga
- Paginación de anime y manga
- Responsive design
